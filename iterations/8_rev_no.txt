Reverse The Number

If the input given is zero display "Output is:0". If the input is less than zero then display "Number should be positive".

Refer the sample input and output.


Sample Input 1 :

Enter The Number:

478

Sample Output  1:
Output is:874


Sample Input  2:

Enter The Number:

-470

Sample Output  2:
Number should be positive


Sample Input  3:

Enter The Number:

34

Sample Output  3:
Output is:43
